1.	List

•	Structure  :-
List is denoted by square brackets []

•	Types of data :-
List is heterogeneous type of data. Means list is capable of holding different type of data.

•	Sequence :-
It is ordered collection.
The benefit of ordered collection is we can perform indexing and slicing on the data.

•	Changeable :-
List is mutable i.e., python will not create a new list if we modify an element in the list.
Different operations like insertion and deletion can be performed on  list.
